# Seekster Test - Backend Developer

## How to runing the app

### On Docker
```bash
$ docker-compose up dev
```

### On CMD
```bash
$ npm install
$ npm run start
# watch mode
$ npm run start:dev
```

#### \*** Server running on http://localhost:3005 \***
